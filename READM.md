## Student Enrollment  Dashboard

## Objective
            This  project  mostly concenrates on student enrollments dashboard using sample educational data and
understand core concepts such as data loading, data transformation, data modeling, visualization, and reporting.


## Dataset
* Excel  Student  Enrollments  data

## Dashboard Features
* load data into power bi 
* clean data in power query 
* Create relations between the table of data models in modeling
* Create measures like Total Students, Course Completion %, and Average Score by using DAX functions 
* Design dashboards and reports in  visualization

# Tools Used
* Power BI
* Power Query
* Data modeling
* Dax
*visualization
